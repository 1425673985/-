# 附近地摊项目迁移指南

由于"附近地摊"和"附近寻厕"是两个不同的项目，建议将"附近地摊"项目文件单独放在一个独立的文件夹中。

## 操作步骤

### 1. 创建新项目文件夹

在文件系统中创建新文件夹（例如在 `/Users/xiongsun/` 目录下）：
- 文件夹名称：`附近地摊`

### 2. 需要复制的文件清单

请将以下文件和文件夹复制到新的"附近地摊"文件夹中：

#### 核心配置文件
- `app.js` - 小程序主逻辑（已更新为附近地摊）
- `app.json` - 小程序配置（已更新为附近地摊）
- `app.wxss` - 全局样式（已更新为橙色主题）
- `sitemap.json` - 站点地图配置
- `project.config.json` - 项目配置（需要更新项目名称为"附近地摊"）
- `project.private.config.json` - 私有项目配置

#### 页面文件
- `pages/index/` - 首页（地图显示及摊位标记）
  - index.js
  - index.json
  - index.wxml
  - index.wxss
- `pages/profile/` - 个人中心
  - profile.js
  - profile.json
  - profile.wxml
  - profile.wxss
- `pages/vendor/` - 商家操作页
  - vendor.js
  - vendor.json
  - vendor.wxml
  - vendor.wxss

#### 云函数
- `cloudfunctions/getNearbyStalls/` - 获取附近摊位云函数
  - index.js
  - package.json
- `cloudfunctions/updateStall/` - 更新摊位信息云函数
  - index.js
  - package.json

#### 文档文件
- `数据库设计说明.md` - 数据库设计文档

#### 其他文件（可选）
- `images/` - 图片资源文件夹（如果需要自定义图标）

### 3. 需要更新的配置

复制文件后，请在新的"附近地摊"项目中更新以下配置：

#### project.config.json
```json
{
  "projectname": "附近地摊",  // 更新项目名称
  "appid": "your-appid-here"  // 使用新的小程序AppID（如果有）
}
```

#### app.js
```javascript
env: 'your-env-id', // 更新为新的云开发环境ID（如果有）
```

### 4. 恢复"附近寻厕"项目（可选）

如果您希望恢复"附近寻厕"项目的原始文件，需要从备份或版本控制系统中恢复以下文件：
- `app.js`
- `app.json`
- `app.wxss`
- `pages/index/` 目录下的所有文件

## 快速操作命令（Mac/Linux终端）

如果您熟悉终端操作，可以使用以下命令：

```bash
# 1. 创建新项目文件夹
cd /Users/xiongsun
mkdir 附近地摊

# 2. 复制核心文件
cp -r 附近寻厕/app.js 附近寻厕/app.json 附近寻厕/app.wxss 附近寻厕/sitemap.json 附近地摊/
cp -r 附近寻厕/project.config.json 附近寻厕/project.private.config.json 附近地摊/

# 3. 复制页面文件
cp -r 附近寻厕/pages 附近地摊/

# 4. 复制云函数
cp -r 附近寻厕/cloudfunctions 附近地摊/

# 5. 复制文档
cp 附近寻厕/数据库设计说明.md 附近地摊/

# 6. 复制图片文件夹（如果有自定义图标）
# cp -r 附近寻厕/images 附近地摊/
```

## 注意事项

1. **项目独立性**：两个项目应该使用不同的小程序AppID和云开发环境ID
2. **数据库**：两个项目使用不同的数据库集合（附近寻厕使用厕所相关集合，附近地摊使用stalls集合）
3. **图标资源**：如果使用了自定义图标，需要将图标文件也复制到新项目中
4. **版本控制**：建议使用Git等版本控制系统管理两个项目

## 验证清单

复制完成后，请确认：
- [ ] 所有页面文件已复制
- [ ] 所有云函数已复制
- [ ] app.json 中的页面路径正确
- [ ] project.config.json 中的项目名称已更新
- [ ] 云开发环境ID已更新（如果使用新的环境）
- [ ] 小程序AppID已更新（如果使用新的AppID）
