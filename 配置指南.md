# 快速配置指南

## 第一步：配置小程序AppID

1. 打开 `project.config.json`
2. 找到 `"appid": "your-appid-here"`
3. 替换为你的小程序AppID

## 第二步：配置腾讯地图API Key

1. 访问 [腾讯位置服务](https://lbs.qq.com/)
2. 注册/登录账号
3. 进入控制台，创建应用
4. 获取Key（API密钥）
5. 打开 `pages/index/index.js`
6. 找到 `const key = 'YOUR_TENCENT_MAP_KEY'`
7. 替换为你的实际Key

## 第三步：配置request合法域名

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入你的小程序后台
3. 开发 -> 开发管理 -> 开发设置
4. 在"服务器域名"中添加：
   - `https://apis.map.qq.com` (request合法域名)

## 第四步：测试运行

1. 使用微信开发者工具打开项目
2. 确保已配置好AppID
3. 点击"编译"按钮
4. 在模拟器中测试功能

## 常见问题

### Q: 搜索不到厕所？
A: 检查以下几点：
- 腾讯地图Key是否正确配置
- request合法域名是否已配置
- 网络连接是否正常
- 如果API不可用，代码会自动使用模拟数据

### Q: 导航功能不工作？
A: 导航功能会依次尝试：
1. 跳转到腾讯地图小程序
2. 使用微信内置打开地图
3. 提示用户手动导航

### Q: 定位权限被拒绝？
A: 用户需要在设置中手动开启定位权限，或者删除小程序重新进入。
