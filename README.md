# 🚻 Nearby Toilets（附近如厕）

一个**快速、直观、只做一件事**的微信小程序：  
在你最需要的时候，**帮你找到最近的公共厕所，并一键导航**。

---

## ✨ 项目简介

**Nearby Toilets（附近如厕）** 是一款基于地理位置的轻量级微信小程序。  
它的设计目标非常简单：

> **用最少的操作，找到离你最近、最合适的公共厕所。**

不做复杂功能，不做冗余页面，只关注「当下、紧急、好用」。

---

## 🧭 核心功能

- 📍 自动获取当前位置
- 🚻 查找附近公共厕所
- 📏 按距离排序显示
- 🗺️ 一键跳转地图进行导航
- ⚡ 快速启动，操作直观

---

## 🛠 技术栈

- **微信小程序**
- **JavaScript**
- 微信定位 API
- 地图 / 导航能力（如腾讯地图）
