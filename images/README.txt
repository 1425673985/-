地图标记图标说明
================

如果需要自定义地图标记图标，请在此目录下添加以下文件：

- toilet-marker.png (推荐尺寸: 80x80px，2倍图)

如果不添加图标，小程序会使用默认的地图标记样式。

图标设计要求：
- 格式：PNG（支持透明背景）
- 尺寸：80x80px（2倍图，实际显示40x40px）
- 文件大小：控制在 5KB 以内（性能优化）
- 颜色：推荐使用红色系（#FF6B6B 或 #FF4757），与应急场景匹配
- 设计：圆形背景 + 厕所图标，与默认POI区分
- 图标：使用简洁的厕所图标（🚻 或简化图形）

设计目标：
1. 与默认地图 POI 做区分（使用圆形设计）
2. 强调"这是厕所"（使用厕所图标）
3. 不影响地图性能（控制文件大小和尺寸）

详细设计指南请参考：MARKER_DESIGN_GUIDE.md
