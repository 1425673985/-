# 公共厕所 Marker 图标设计指南

## 设计目标
1. 与默认地图 POI 做区分
2. 强调"这是厕所"
3. 不影响地图性能

## 图标设计方案

### 方案一：简洁图标（推荐）
**设计要点：**
- 形状：圆形背景 + 厕所图标
- 尺寸：80x80px（2倍图，适配高分辨率屏幕）
- 颜色：红色系（#FF6B6B 或 #FF4757），与应急场景匹配
- 图标：使用简洁的厕所图标（🚻 或简化图形）
- 背景：白色圆形，带红色边框

**视觉特点：**
- 圆形设计，与默认POI的方形/水滴形区分
- 红色系，在地图上醒目且与应急场景匹配
- 简洁图标，快速识别

### 方案二：水滴形图标
**设计要点：**
- 形状：水滴形（类似定位点）+ 厕所图标
- 尺寸：80x80px
- 颜色：橙色系（#FF8E53），温暖且醒目
- 图标：厕所图标位于水滴形中心
- 背景：渐变橙色

**视觉特点：**
- 水滴形与定位点相似，但颜色不同
- 橙色系，与红色系区分，避免与用户位置混淆

### 方案三：方形图标（最清晰）
**设计要点：**
- 形状：圆角方形 + 厕所图标
- 尺寸：80x80px
- 颜色：深蓝色系（#4A90E2），专业且清晰
- 图标：厕所图标 + "WC"文字
- 背景：白色圆角方形，蓝色边框

**视觉特点：**
- 方形设计，与圆形POI明显区分
- 文字标识，最直观表达"这是厕所"

## 推荐方案：方案一（简洁图标）

### 设计规范
- **尺寸**：80x80px（2倍图，实际显示40x40px）
- **格式**：PNG（支持透明背景）
- **颜色**：
  - 背景：白色圆形
  - 边框：2px，红色 #FF6B6B
  - 图标：红色 #FF4757
- **图标**：使用 Unicode 厕所图标 🚻 或简化图形
- **阴影**：轻微阴影，增强层次感

### 文件命名
- `toilet-marker.png` - 标准图标（80x80px）
- `toilet-marker-highlight.png` - 高亮图标（当前选中，可选）

## 性能优化建议

1. **图标尺寸控制**
   - 使用 80x80px（2倍图），避免过大影响性能
   - 文件大小控制在 5KB 以内

2. **图标数量控制**
   - 只显示当前选中的厕所 marker
   - 避免同时显示多个 marker

3. **图标缓存**
   - 图标文件放在本地，避免网络加载
   - 使用 PNG 格式，压缩优化

4. **渲染优化**
   - 使用 `alpha: 1.0` 避免透明度计算
   - 避免频繁更新 marker 位置
